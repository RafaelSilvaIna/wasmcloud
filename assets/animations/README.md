# 🎬 PipoCine - Animações SVG

Esta pasta contém animações SVG profissionais usando SMIL (Synchronized Multimedia Integration Language) para apresentações interativas.

## 📁 Arquivos

### connection-paths.svg
Animação de linhas conectando plataformas com:
- Paths animados com gradiente
- Pacotes de dados fluindo entre pontos
- Grid de fundo sutil
- Efeito glow nos elementos

### streaming-waves.svg
Ondas de streaming com:
- Múltiplas ondas animadas com diferentes timing
- Partículas flutuando ao longo das ondas
- Barras verticais representando dados
- Gradientes suaves

### orbit-rings.svg
Sistema orbital com:
- Anéis girando em diferentes velocidades
- Plataformas orbitando em paths elípticos
- Linhas conectando centro aos pontos
- Efeito de pulso no centro

### particle-network.svg
Rede de partículas com:
- Nós (plataformas) conectados por linhas
- Partículas de dados fluindo entre nós
- Nós secundários flutuantes
- Efeito de pulsação nos nós principais

### pulse-rings.svg
Anéis pulsantes com:
- Múltiplos anéis expandindo do centro
- Timing escalonado para efeito contínuo
- Linhas rotativas ao redor
- Centro pulsante

### data-stream.svg
Fluxo de dados com:
- Linha principal com gradiente animado
- Pacotes de dados retangulares em movimento
- Partículas de fundo piscando
- Movimento contínuo da esquerda para direita

## 🚀 Uso

As animações são referenciadas como imagens SVG e executam automaticamente:

```html
<img src="/assets/animations/connection-paths.svg" alt="" loading="lazy">
```

## 🎨 Cores

Todas as animações usam a paleta da marca PipoCine:
- **Primary**: `#e50914` (vermelho Pipocine)
- **Secondary**: `#ff6b6b` (rosa/vermelho claro)
- **Background**: Transparente ou escuro

## 📱 Responsivo

As animações são escaláveis e mantêm a qualidade em qualquer tamanho de tela.

## ⚙️ Performance

- Animações otimizadas usando SMIL nativo
- Não requer JavaScript adicional
- Lazy loading recomendado
- GPU-accelerated onde suportado
